Preguntas a resolver:

  1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

    2. Ambos lenguajes comparten sintaxis similar y su orientacion en objetos. Sin embargo tienen mas diferencias tales como:

    1. Java es un lenguaje de proposito general, mientras que javascript esta diseñado para el desarrollo web
    2. El Java se compila en bytecode y se ejecuta en una maquina virtual, mientras que javascript se ejecuta directamente en el navegador web o en el servidor a traves del Node.js
  2. ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)
    1. getDate(): Te da el día del mes (del 1 al 31).
    2. getMonth(): Te da el mes (del 0 al 11).
    3. getFullYear(): Te da el año (año completo).
    4. getHours(): Te da las horas (del 0 al 23).
    5. getMinutes(): Te da los minutos (del 0 al 59).
    6. getSeconds(): Te da los segundos (del 0 al 59).
    7. etMilliseconds(): Te da los milisegundos (del 0 al 999).
  3. ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

    3. unshift(): Añade un elemento al principio del arreglo

  4. ¿Cómo se declara una variable con alcance local dentro de una función?

    5. Se puede declarar una variable de alcance local con ayuda de la palabra clave "let" o "const"

  5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    6. Hay un riesgo de colisiones y errores. Esto debido a que dichas variables globales pueden ser modificadas en cualquier parte del programa. Dichas variables son utiles si quieres modificar una misma variable durante todo el programa
